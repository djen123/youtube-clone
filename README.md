Moder ,responsive youtube  style video built using html ,css and javascript,
project allows users to search for youtube videos,filter results ,brows categoies such as music ,game and etc
## Technology Used
HTML,CSS3,Javascript,Youtube Data API v3


How it works 
- user enters a search term
- App calls the YouTube API
- Results are then displayed as video cards
- Clicking a card loads an embedded player
- Sidebar categories  when click triggers instant searches
- filters reorder the results

further imporvements can be done by adding more features such as user login,search history etc
